[译] JSP 和 JSPX 的区别

首先，JSPX 是完成符合 XML 格式的 JSP。

进一步，JSPX 强制分离了 JS 和 XML 代码；而 JSP 在 XML/HTML中 嵌入 Java 逻辑，这不利于测试。

JSPX 有利于及早纠正错误。

多数情况下，JSPX 语法更简单。

但涉及动态内容、复杂表达式或执行逻辑时，JSP 显然更胜一筹。
